1. DESCRIPTION
----------------------

Dans le cadre de cette partie, nous avons utilisé l'outil docker afin de déployer notre serveur Gitea.

1.1 DEFINITION
--------------
Gitea est un service de développement logiciel tout-en-un, simple et auto-hébergé. Il inclut l'hébergement Git, la revue
de code, la collaboration en équipe, le registre de paquets et le CI/CD.

2. OUTILS UTILISES
------------------

- SSH 
- De vscode pour éditer nos différentes configurations
- D'un serveur nginx afin de l'utiliser comme reverse-proxy
- D'un OS (Linux) (afin d'exécuter les différentes commandes administratives et de la  gestion de permission)
- De OpenSSL afin de générer nos certificats auto-signés
- De AWX pour synchroniser Gitea

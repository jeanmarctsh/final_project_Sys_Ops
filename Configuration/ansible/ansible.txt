Projet : Ansible Project
Description : configuration d'iventaire (format yaml) et de différents playbook pour mon infrastructure avec 2 machines clientes
Lien GitHub : https://github.com/jeanmarctsh/ansible_project
Branche principale : patchlab

Dans le cadre de cette partie nous avons utilisé ansible afin de configurer nos deux machines clientes. 

Pourquoi ai-je utilisé ansible?
Compte tenu des spécificités de mon ordinateur, j'ai utilisé une VM ansible pour mieux gérer les ressources de mon ordinateur: 
c'est-à-dire RAM et CPU. Afin de configurer les machines clientes et transférer le dossier compresser au serveur Gitea avec rsync.

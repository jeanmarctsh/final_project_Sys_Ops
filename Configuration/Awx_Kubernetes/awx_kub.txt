1. Description
--------------

De nos jours, avoir une vision globale sur un système informatique devient de plus en plus nécessaire.
il est important de noter que la vision globale ne suffit pas, il faut savoir organiser, planifier, exécuter, checker les différents logs.

Dans le cadre de notre travail, nous avons utilisé AWX-OPERATOR, qui est un outil open source qui permet d'effectuer ces différentes tâches.


2. Outils utilisés
------------------

- OS: LINUX
- KUBERNETES
- VS_CODE
- SSH
- HYPERVISEUR type 2 : vmware_workstation
- PROTOCOLE RSYNC
- DOCKER
